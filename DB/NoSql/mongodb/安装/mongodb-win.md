1. 管理员 权限 cmd


1. Start MongoDB Community Edition as a Windows Service

2. Stop MongoDB Community Edition as a Windows Service

3. Remove MongoDB Community Edition as a Windows Service
   `sc.exe delete MongoDB`

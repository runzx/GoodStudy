# WSL 维护

0. 查看  
   `wsl -l`

1. 停止子系统  
   `wsl -t Alpine`

# backup 先停止子系统之后再备份

0. 备份  
   `wsl --export Alpine d:\bak\Alpine-202012.tar`
1. 删除子系统  
   `wsl --unregister Alpine`
2. 还原  
   `wsl --import Alpine c:\WSL d:\bak\Alpine-202012.tar`  
   2.6G /home 330M, /root 950M,/data 820M /usr 660M (bin 415M)

3. 如果系统里已有，要先注消: `wsl --unregister Alpine`

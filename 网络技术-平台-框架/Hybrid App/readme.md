## Hybrid App 本质
原生Native App中，使用WebView作为容器，承载一个web 页面  

IOS 要求，原生ui与web ui 2/8  

### JSBridge: 接口、桥梁
原生（Native）和 web 端的双向通讯层（跨语言解决方案）  

  

### WebView
是一个基于webkit引擎、展现web页面的控件。 
直接使用html文件（网络上或本地assets中）作布局  
可和JavaScript交互调用
